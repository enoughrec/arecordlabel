[EN] 3rd edition of the Falésia compilation, devoted to Portuguese projects surrounding the dark ambient genre neighbourhood. Participation from the following projects: <a href="http://www.myspace.com/dymanofficialspace">Dyman</a>, <a href="http://www.myspace.com/alexmoorept">Alex Moore</a>, <a href="http://www.myspace.com/thealiceinwonderlandsyndrome">Alice In Wonderland Syndrome</a>, <a href="http://www.myspace.com/sektor304">Sektor 304</a>, <a href="http://www.myspace.com/ambiansu">Ambiansu</a>, Canau, <a href="http://www.myspace.com/mu73">João Farelo</a>, <a href="http://www.myspace.com/dsm666">DSM666</a>, <a href="http://www.myspace.com/hollenfahrt">Höllenfahrt</a>, <a href="http://www.myspace.com/hugopaquete">Hugo Paquete</a> and <a href="http://www.myspace.com/pif314">P.I.F.(3,14...)</a>. Artwork by <a href="http://www.myspace.com/rastodetinta">André Coelho</a>.
