[EN] Third edition of the Dark Vault compilation, 17 years after the first edition, 13 since the last, we continue the legacy of gathering tracks of cinematic dark ambient vein from our active artist roster and packaged them for you in a compilation format. We had so many submissions we divided this release into three parts and mixed the tracks together for a free flowing continuous listening experience. Tracks by <a href="https://soundcloud.com/nightech">Plour</a>, <a href="https://www.okammusic.com/">OKAM</a>, <a href="https://mpex.bandcamp.com/">M-PeX</a>, <a href="https://esc-laboratory1.bandcamp.com/">In This Place There Is No End</a>, <a href="https://soundcloud.com/audiocompress">Audio Compress</a>, <a href="https://www.moan.pl/">MOAN</a>, <a href="https://thermidor.bandcamp.com/">Thermidor</a>, <a href="https://x-rtb.net/">Xalm Retribution</a>, <a href="https://janrobbe.bandcamp.com/">Diagnostic</a>, <a href="https://curtainsinthecorner.bandcamp.com/">Picture in the Room</a>, <a href="https://soundcloud.com/musicknut">Knut</a>, <a href="https://tpolm.org/~ps">ps</a> feat. <a href="https://www.facebook.com/antonio.boieiro">António Boieiro</a>, <a href="https://myspace.com/0xyg3n">Oxygen</a>, <a href="https://soundcloud.com/rsvaudio">r s v</a>, <a href="https://dreammetaphor.net/">Systemic Failure</a>, <a href="https://soundcloud.com/vysehrad">Vysehrad</a>, <a href="https://www.youtube.com/channel/UCBruci-3AIThKgjWEXgHnlg">Singularity Observatory</a>, <a href="https://captiveportal.bandcamp.com/">This Communication</a>, <a href="https://alexmason17.bandcamp.com/">Alex Mason</a>, <a href="https://sensor3.bandcamp.com/">Sensor</a>, <a href="https://twitter.com/mantratronic">Mantratronic</a> and <a href="https://thesilenceindustry.bandcamp.com/">The Silence Industry</a>. Tivemos tantas submissões que dividimos a edição em 3 partes, cada uma misturando os temas numa sequência pensada para a audição continua.
