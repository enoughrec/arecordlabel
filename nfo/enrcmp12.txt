[EN] Compilation of Portuguese projects active in 2008 surrounding the industrial genre, inspired by the <a href="http://industrialpt.net">industrialpt association</a>. Limited digipack physical edition of 500 copies will come out through <a href="http://thisco.net">Thisco Records</a> in early 2009. Participating projects in order of tracklist appearance are: <a href="http://myspace.com/z0te">Z.O.T.E.</a>, <a href="http://tpolm.org/~ps/">ps</a> vs <a href="http://myspace.com/sektor304">Sektor 304</a>, <a href="http://myspace.com/wastedisposalmachine">Waste Disposal Machine</a>, <a href="http://myspace.com/controlaltdeus">Control Alt Deus</a>, <a href="http://myspace.com/aktivehate">aktiv[E|H]ate</a>, <a href="http://myspace.com/7h31nc1d3n7">1N.C1.D3.N7</a>, <a href="http://myspace.com/scifiindustries">Sci-Fi Industries</a>, <a href="http://myspace.com/kronosrock">Kronos</a>, <a href="http://myspace.com/violetandthemutants">Violet And The Mutants</a>, <a href="http://myspace.com/mikroben">Mikroben Krieg</a>, <a href="http://myspace.com/structura">Structura</a>, <a href="http://myspace.com/stuka666">Stuka</a>, <a href="http://myspace.com/dsm666">DSM666</a>, <a href="http://myspace.com/ummebm">U.M.M.</a> and <a href="http://myspace.com/oocito2">Oocito II</a>. Artwork by <a href="http://homemdopacote.wordpress.com">André Coelho</a>.

[PT] Compilação de projectos Portugueses activos em 2008 em géneros arredores ao industrial. Fortemente inspirada pela <a href="http://industrialpt.net">associação industrialpt</a>. Edição fisica em digipack limitada a 500 copias co-editada pela <a href="http://thisco.net">Thisco Records</a> em inicios de 2009. Projectos participantes incluem <a href="http://myspace.com/z0te">Z.O.T.E.</a>, <a href="http://tpolm.org/~ps/">ps</a> vs <a href="http://myspace.com/sektor304">Sektor 304</a>, <a href="http://myspace.com/wastedisposalmachine">Waste Disposal Machine</a>, <a href="http://myspace.com/controlaltdeus">Control Alt Deus</a>, <a href="http://myspace.com/aktivehate">aktiv[E|H]ate</a>, <a href="http://myspace.com/7h31nc1d3n7">1N.C1.D3.N7</a>, <a href="http://myspace.com/scifiindustries">Sci-Fi Industries</a>, <a href="http://myspace.com/kronosrock">Kronos</a>, <a href="http://myspace.com/violetandthemutants">Violet And The Mutants</a>, <a href="http://myspace.com/mikroben">Mikroben Krieg</a>, <a href="http://myspace.com/structura">Structura</a>, <a href="http://myspace.com/stuka666">Stuka</a>, <a href="http://myspace.com/dsm666">DSM666</a>, <a href="http://myspace.com/ummebm">U.M.M.</a> e <a href="http://myspace.com/oocito2">Oocito II</a>.
